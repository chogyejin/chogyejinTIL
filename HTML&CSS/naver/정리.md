# display

display:block -> 부모 width 만큼 차지 안 해도 다음 줄로 넘어간다(margin으로 채워짐, 너비 100%).
display:inline-block -> 자기 컨텐츠나 너비만큼만 공간 차지
span 태그는 display:inline
