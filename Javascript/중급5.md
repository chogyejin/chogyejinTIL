# Class(ES6에 추가)

- 클래스 내에 선언한 메서드는 만들어진 객체의 프로토타입에 저장됨.
  (생성자 함수는 객체 내부에 저장)
- 생성자 함수 이용했을 때 new 키워드 붙이지 않고 객체 만들면 undefined 반환, 클래스는 typeError 반환(constructor가 class임이 명시되어 있다.)

```
class User {
    constructor(name,age){
        this.name = name;
        this.age = age;
    }
    showName(){
        console.log(this.name);
    }
}

const tom = new User("Tom",19);
```
